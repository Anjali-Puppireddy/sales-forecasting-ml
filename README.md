# Sales Forecasting Using Machine Learning

## Project Overview
Built a sales forecasting model using historical sales data to predict future demand and support business decision-making.

## Tech Stack
- Python
- SQL
- Pandas
- Scikit-learn
- Random Forest

## Key Features
- Time-based train-test split to prevent data leakage
- Feature engineering (lag features, moving averages)
- Hyperparameter tuning using GridSearchCV
- Model evaluation using RMSE

## Business Impact
- Reduced stockouts and overstocking
- Improved demand planning and inventory management
- Enabled data-driven forecasting decisions
